Bootstrap Progress Bars

Basic Progress Bar

A progress bar can be used to show a user how far alone he/she is in a process. Bootstrap provides several types of progress bars. A default progress bar in bootstrap looks like below one :

70% Complete

To create a dafault progress bar, add a .progress class to a <div> element.

Note: Progress bars are not supported in IE9 and earlier (because they use CSS3 transitions and animations to achieve some of their effects).

Note: To help improve accessability for people using screen readers , you should include the aria-* attributes.
