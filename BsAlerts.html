Bootstrap alerts

Alerts

Bootstrap provides an easy way to create pre-defined alert messages. Alerts are created with .alert class, followed by one of the four contextual classes .alert-success,.alert-info, .alert-warning or .alert-danger

Success! Indicates a successful or positive action.
Info! Indicates a neautral informative change or action.
Warning! Indicates a warning that might need attention.
Danger! Indicates a dangerous or potentially negative action.

Alert Links

Add the alert-link class to any links inside the alert box to create "matching colored links:"
