1. Bootstrap uses HTML elements and css properties that require HTML5 doctype
  2. Bootstrap 3 is designed to be responsive to mobile devices
  3. Mobile first styles are part of the core framework
  4. Meta tag inside the head head element is required to ensure
    a. The proper rendering and
    b. Touch zooming
  5. The width='device-width' part sets the width of the page to follow the screen-width of the device.
  6. Device width will vary depending upon the device
  7. initial-scale=1 part sets the initial zoom level when the page is first loaded by the browser
