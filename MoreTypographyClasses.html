Class Description Example
.lead Makes a paragraph stand out
.small Indicates smaller text (set to 85% of the size of the parent)
.text-left Indicates left-aligned text
.text-center Indicates center-aligned text
.text-right Indicates right-aligned text
.text-justify Indicates justified text
.text-nowrap Indicates no wrap text
.text-lowercase Indicates lowercased text
.text-uppercase Indicates uppercased text
.text-capitalize Indicates capitalized text
.intialism Displays the text inside an <abbr> element slightly smaller font size.
.list-unstyled Removes the default list-style and left magin on list items (works on both <ul> and <ol>). This class only applies to immediate children list items (to remove the dafult list-style from any nested lists, apply this class to any nested lists as well)
.list-inline Places all list items on a single line
.dl-horizontal Lines up the terms (<dt>) and descriptions (<dd>) in <dl> elememts side-by-side. Start off like default <dl>s, but when the browser window expands, it will line uo side-by-side.
.pre-scrollable Makes a <pre> element scrollable
